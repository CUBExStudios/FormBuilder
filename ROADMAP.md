# Form Builder Roadmap

## Phase 1 - MVP
- Form creation UI
- Basic elements (text, number, date, rating)
- Form sharing + embed
- Response storage
- CSV export

## Phase 2 - Advanced Features
- Conditional logic
- Themes
- File uploads
- Payments
- Analytics

## Phase 3 - Integrations
- Webhooks
- Zapier/N8N
- Slack, Discord

## Phase 4 - Pro Features
- Templates marketplace
- Team accounts
- AI question generator
