# Contributing to <PROJECT NAME>

Thank you for your interest in contributing! ❤️

## Ways to Contribute
- Bug fixes
- New features
- Documentation
- UI/UX improvements
- Discussions & idea proposals

## Pull Request Process
1. Fork the repository
2. Create a branch: `feature/my-feature`
3. Add your changes
4. Ensure code works locally
5. Open a pull request

We will review PRs within 48–72 hours.
